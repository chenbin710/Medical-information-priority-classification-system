# Medical Information Priority Classification System

这是一个 Vibe Coding 作业项目，用于医药资讯优先级分类。

## 技术栈
- Next.js 15 + TypeScript
- shadcn/ui + Tailwind CSS
- GitHub 协作

## 如何运行
1. git clone https://github.com/chenbin710/Medical-information-priority-classification-system.git
2. cd medical-priority
3. npm install
4. npm run dev
5. 打开 http://localhost:3000

## 当前功能
- 首页表格显示硬编码/ mock 新闻
- 后续：接入中台 API、优先级判断、二次复核等
